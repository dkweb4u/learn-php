Inheritence

Its access another class methods using the "extends" keyword

        class User{
            public $name;
            public $email;
            protected $status = 'active';
        
            public function __construct($name, $email){
                $this->name = $name;
                $this->email = $email;
            }
        
            public function login(){
                echo "Logged is successfully";
            }
        
            public function getprivatValue(){
                return $this->status;
            }
        }
        
        class Admin extends User{  // keyword is very important for extends
            public $level;
            public  function __construct($name, $email, $level){
                $this->level = $level;
                parent::__construct($name,$email);   // call parent class and method
            }
        
        }
        
        $admin1 = new Admin("dinesh","dk@gmail.com",5);
        
        print_r($admin1);

Polymorphism

Its override the methods from parent class to child class

Below example two same method name is there its called parent child method 

    class User{
        public $name;
        public $email;
        protected $status = 'active';
    
        public function __construct($name, $email){
            $this->name = $name;
            $this->email = $email;
        }
    
        public function login(){
            echo "Logged is successfully";
        }
    
        public function getprivatValue(){
            return $this->status;
        }
    }
    
    class Admin extends User{  // keyword is very important for extends
        public $level;
        public  function __construct($name, $email, $level){
            $this->level = $level;
            parent::__construct($name,$email);   // call parent class and method
        }
    
        // Its called Polymorphism 
        public function login(){            // same  method for parent
            echo "Admin Logged is successfully";
        }
    
    }
    
    $admin1 = new Admin("dinesh","dk@gmail.com",5);
    
    print_r($admin1->login());

Static method in class

Its used for no need to call in "new class" instead of "::" classname::value you can class directly

    class Maths{

        public static $pi = 3.14;
    
        public static function add(...$num){
            return array_sum($num);
        }
    
    }
